# Requerimientos del Sistema

Para garantizar el correcto funcionamiento del software de gestión JAC, se deben cumplir los siguientes puntos:

## Requerimientos Funcionales (Lo que hace el software)
* **RF1:** El sistema debe permitir el registro de Juntas de Acción Comunal por municipio.
* **RF2:** El sistema debe calcular automáticamente el valor de la "Minga" (aporte comunitario).
* **RF3:** Generación de reportes consolidados en formato PDF/Web.

## Requerimientos de Software (Lo que necesita para correr)
* **Lenguaje:** PHP 8.x o superior.
* **Base de Datos:** MySQL / MariaDB.
* **Servidor Local:** XAMPP.
* **Dependencias:** MkDocs (para la documentación).
